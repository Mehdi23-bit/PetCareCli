#  PetCare CLI

A command-line pet management app built with Python.

## Features
- Add, list, search, update, delete pets
- Export to CSV
- Backup data
- Colorful terminal interface

## Installation
```bash
git clone https://github.com/Mehdi23-bit/PetCareCli
cd PetCareCli
pip install -r requirements.txt
python petcare.py
```
